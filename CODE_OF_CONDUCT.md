# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) (v2.1).

In short: be respectful, assume good intent, and keep discussion constructive. Harassment, discrimination, and personal attacks are not tolerated.

To report unacceptable behavior, email **conduct@didit.me**. Reports are handled confidentially.
